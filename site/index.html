🔧 安裝說明

macOS

  1. 下載對應架構的 .dmg 檔案
  2. 雙擊打開 DMG，將 ABC管理系统 拖到 Applications 資料夾
  3. 首次開啟會被 Gatekeeper 阻擋 → 右鍵 → 打開 → 確認打開
  4. 或在終端執行：xattr -cr /Applications/ABC管理系统.app
  5. 應用程式會自動檢查更新

Windows

  1. 下載 .exe 安裝程式
  2. 執行安裝程式，依照指示完成安裝
  3. 從開始選單啟動 ABC管理系统
